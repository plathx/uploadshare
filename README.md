# uploadshare

ไฟล์อัปโหลด & แชร์ไฟล์ได้ผ่านเว็บไซต์ หลายแพลตฟอร์มการอัปโหลด

## 🚀 เริ่มต้นอย่างรวดเร็ว (3 ขั้นแค่นี้!)

### Windows
```bash
START.bat
```

### Mac/Linux
```bash
chmod +x START.sh
./START.sh
```

### Manual
```bash
npm install
npm start
```

จากนั้นเปิด: **http://localhost:3000** ✨

## ✨ Version 2.0.1 - ทั้ง 8 บริการทำงาน!

## 🚀 สิ่งที่ได้ทำการแก้ไข (Version 2.0)

### ✅ แก้ไขบัค CORS
- เปลี่ยนจาก XMLHttpRequest เป็น Fetch API
- เพิ่ม AbortController สำหรับควบคุมการยกเลิกอัปโหลด
- ปรับปรุงการจัดการข้อผิดพลาดในแต่ละบริการ
- เพิ่มแสดงความคืบหน้าของการอัปโหลด
- **✨ NEW**: Express Server Proxy สำหรับทั้ง 8 บริการ

### 🎯 บริการที่รองรับ (8 บริการ)
1. **Gofile** - อัปโหลดไม่จำกัด (แนะนำ) ✅ ทำงาน
2. **Catbox** - 200 MB, ถาวร ✅ ทำงาน
3. **Litterbox** - 1 GB, 3 วัน (72 ชม.) ✅ ทำงาน
4. **Tmpfiles** - 5 GB, 60 นาที ✅ ทำงาน
5. **Uguu** - 128 MB, 24 ชม. ✅ ทำงาน
6. **File.io** - 2 GB, 1 ครั้งทิ้ง ✅ ทำงาน
7. **Pomf** - 512 MB, ถาวร ✅ ทำงาน
8. **0x0.st** - 512 MB, 30-365 วัน

## 📝 หมายเหตุเกี่ยวกับ CORS

บางบริการอาจยังคงแสดงข้อผิดพลาด CORS เนื่องจากนโยบายความปลอดภัยของเบราว์เซอร์ ตัวเลือกในการแก้ไข:

### ตัวเลือก 1: ใช้ Gofile เพียงอย่างเดียว (ง่ายที่สุด)
- Gofile ทำงานได้ดีที่สุดกับเบราว์เซอร์โดยไม่มี CORS
- เป็นตัวเลือกที่ดีที่สุดสำหรับส่วนใหญ่ของผู้ใช้

### ตัวเลือก 2: เรียกใช้เซิร์ฟเวอร์พร็อกซี
สร้างไฟล์ `proxy.js`:

\`\`\`javascript
// หากต้องการใช้บริการอื่นๆ ให้ตั้งค่าเซิร์ฟเวอร์พร็อกซี Node.js
const express = require('express');
const busboy = require('busboy');
const fetch = require('node-fetch');
const FormData = require('form-data');

const app = express();

app.post('/proxy/:service', (req, res) => {
  const { service } = req.params;
  // ... ดูเอกสารเพิ่มเติม
});
\`\`\`

### ตัวเลือก 3: ใช้บริการที่รองรับ CORS

## 🏃 วิธีการใช้

## 🐛 การแก้ไขปัญหา

### เมื่ออัปโหลดใช้ไม่ได้
1. ลองใช้บริการอื่น (เลื่อนไปที่บริการข้างไป)
2. ตรวจสอบว่าขนาดไฟล์ไม่เกินขีดจำกัด
3. ลองเปิด DevTools (F12) เพื่อดูข้อผิดพลาด

### เมื่อเห็น CORS Error
- เนื่องมาจากนโยบายความปลอดภัยของเบราว์เซอร์
- ลองใช้ Gofile (แนะนำ) หรือ File.io
- หากจำเป็นต้องใช้อื่น ให้ตั้งค่าเซิร์ฟเวอร์พร็อกซี
